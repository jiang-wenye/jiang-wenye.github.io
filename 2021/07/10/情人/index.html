&emsp;&emsp;我知道，女人美不美，不在衣装服饰，不在美容修饰，不因为施用的香脂价钱贵不贵，穿戴珍奇宝物、高价的首饰之类。

//
&emsp;&emsp;我自以为我在写作，但事实上我从来就不曾写过，我以为我在爱，但我也从来不曾爱过，我什么也没有做，不过是站在那紧闭的门前等待罢了。
//
&emsp;&emsp;在我头脑里，她的皮肤的气味，早已没有，不存在了，在我的眼里，她眼睛的颜色也早已无影无踪。那声音，我也不记得了，有时，我还能想起傍晚那种带有倦意的温煦。那笑声，是再也听不到了，笑声，哭声，都听不到了。
//
&emsp;&emsp;这种悲戚忧伤本来是我所期待的，我原本就在悲苦之中，它原本就由我而出。
//
&emsp;&emsp;她是让贫穷给活剥了的母亲，或者她是这样一个女人，在一生各个时期，永远对着沙漠，对着沙漠说话，对着沙漠倾诉，她永远都在辛辛苦苦寻食糊口，为了活命，她就是那个不停地讲述自己遭遇的玛丽·勒格朗·德·鲁拜，不停地诉说着她的无辜，她的节俭，她的希望。
//
&emsp;&emsp;他的英雄气概，那就是我，他的奴性，那就是他父亲的金钱。
//
&emsp;&emsp;所有这些历史事件尽管是那么可怕，而她却超越于历史之上，永远在那条街上匆匆走过。
//
&emsp;&emsp;照这种照片的人我见过不少，拍出的照片几乎一样，惊人地酷似。不仅因为年衰人老而彼此相像，而是因为人像都被修饰描绘过，永远都是这样，颜面上的特征，如果拍出来的话，经过这样修饰，也就抹去看不见了。人的面目经过这样一番修饰，才能正面迎对永恒，人的面貌经过橡皮涂改，一律变得年轻了。

读后感

&emsp;&emsp;第一条摘录出自《情人》的首段，上一本在开头就能牢牢地抓住我的小说还是《洛丽塔》中的“生命之光”、“欲望之火”。衰老的杜拉斯凝望着自己，审视着她少女时代的爱欲。
&emsp;&emsp;
&emsp;&emsp;年少的女主人公出身贫寒的家庭，母亲溺爱兄长，而兄长却只知索取、不知感激回馈，甚至沉溺于赌博，对家人施暴，善良温柔的小哥哥是法国少女的家庭中最美好的存在，但这样的小哥哥却英年早逝。她对破碎的家庭感到心碎而绝望。
&emsp;&emsp;
&emsp;&emsp;华裔男人在码头对少女一见钟情，少女最初或许只是因为男人不俗的衣装与车子爱上男人的金钱。男人与少女交欢，一起沉沦在爱欲之中。少女付出童贞，男人付出金钱。少女需要用钱来接济家人的生活，而家人却默许少女的行为。华裔与白人交往需要背负非常大的压力，男人畏惧周遭的目光，更畏惧身为封建家庭大家长的父亲，因为怕被父亲断掉所有的经济来源，他只让少女做他隐秘的情人。男人或许想过他与少女的未来，所以他和少女的家人见了面。少女的家人在酒店表现得傲慢而粗鲁，鄙视男人的肤色，却又贪恋男人的金钱。少女虽然爱着男人，但也始终因为肤色而怀有偏见。男人虽然一直在为少女的物质生活买单，却从未在这段关系中处于支配地位，他英俊潇洒衣着光鲜，但也始终怀有深深的自卑。
&emsp;&emsp;
&emsp;&emsp;小说中虽然从未细细刻画过男人的家庭，但可以想象的是，在压抑的封建家庭中，华裔男人和法国少女一样拥有一个孤独的灵魂。每一次爱欲涌动，或许都是在向对方谋求爱，谋求温暖。
&emsp;&emsp;
&emsp;&emsp;华裔男人是软弱的，他最终还是和少女分手，受父母之命娶了一个从未见过的女人。少女做出的决定在我看来也足以影响她的一生。她登上了回到法国的渡轮，离开了无爱的家庭，奔向富饶的法国，或许也是奔向光明的未来。
&emsp;&emsp;
&emsp;&emsp;梁家辉的电影中，华裔男人在码头默默向少女告别，不消言说，也令人相信多年后男人对待女人就像小说结尾一样“他对她说，和过去一样，他依然爱她，他根本不能爱她，他说他爱她将一直爱到他死”。
&emsp;&emsp;
&emsp;&emsp;但故事中的平行世界呢？或许男人在婚姻之后获得了新的恋爱，或许没有，但他再也不敢反抗来自父亲的压力，即便只是心里微漠的渴望。
